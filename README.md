Ansible Role Nginx Lb
======================================

This role will configure running nginx instance as loadbalancer.

Requirements
------------

None.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------
```
- hosts: all
  roles:
    - role: vundb-nginx-lb
```
